# Assignment 1 - Circular Buffer Implementation

## Description

This project implements a Circular Buffer in C without using any built-in data structures. The goal is to understand how circular buffers work and how they differ from linear buffers.

## Features

* Buffer initialization
* Writing data into the buffer
* Reading data from the buffer
* Handling overflow and underflow conditions

## Program Behavior

* The program reads a name from standard input.
* It appends the string "CE-ESY" to the name.
* The result is stored in a circular buffer.
* The data is then read back from the buffer and printed.
* The buffer is emptied after reading all data.

## Notes

* The buffer size is intentionally smaller than the input size to test overflow behavior.
* The implementation is done manually without using ready-made data structures.
